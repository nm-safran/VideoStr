How to add a new student

How to add a new course

Edit data.js ‚Üí add a new object to the COURSES array with a unique id ‚Üí add lessons with driveId ‚Üí assign course ID to students in USERS ‚Üí commit.

How to remove a student's access to a course

Find the student in USERS ‚Üí remove the course ID from their courses array ‚Üí commit. The student won't see that course on next login.
